# React + TypeScript + Vite Todo App

A responsive, feature-rich Todo List application built using React class components, TypeScript, and styled using React Bootstrap. 

## Features

- **Add Tasks:** Quickly add new tasks to your checklist.
- **Edit Tasks:** Modify existing tasks inline using prompts.
- **Delete Tasks:** Remove completed or unwanted tasks from the list.
- **Responsive Layout:** Sleek styling powered by React Bootstrap columns and list groups.
- **State Immutability:** Uses pure immutable state update operations (e.g., mapping over arrays to modify items) for optimal React performance and reliable rendering.
- **TypeScript Checked:** Written in TypeScript (`.tsx`) with strict typing to ensure type safety.

## Getting Started

Follow these steps to run the application locally:

### 1. Install Dependencies
Run the following command in the project root directory:
```bash
npm install
```

### 2. Start the Development Server
Launch the local development environment:
```bash
npm run dev
```

### 3. Build for Production
Bundle the application for production:
```bash
npm run build
```

## Tech Stack

- **Framework/Build Tool:** [Vite](https://vite.dev/)
- **Language:** [TypeScript](https://www.typescriptlang.org/)
- **UI Framework:** [React 19](https://react.dev/)
- **Styling:** [Bootstrap 5](https://getbootstrap.com/) & [React Bootstrap](https://react-bootstrap.github.io/)
